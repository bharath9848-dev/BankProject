# Bank Management System (Java)
## Description
This project is a simple Bank Management System built using Java and JDBC.
It allows users to create accounts, deposit money, withdraw money,
check balance,view account details and Transcation history.
## Features
- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- Delete Account
- Transcation history
- View All Accounts
## Technologies Used
- Java
- Object Oriented Programming
- ArrayList
- Scanner
